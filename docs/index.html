What is it?

Ts.ED is a framework on top of Express to write your application with TypeScript (or ES6). It provide a lot of decorators and guideline to write your code.

Rest API

Create your Rest API easily and create different version path of your API

Configuration

Didn't waste your time with configuration, the server is preconfigured to start quickly!

Class based

Define class as Controller, Model, Service (DI), Filter, Middleware, Converter etc...

Decorators

A lot of decorators are provided to structure your code and define route and method.

Testing

Test or not to test? Isn't a question. Ts.ED embed some features to test your code!

And more

Find some guide to integrate Swagger, AJV validation, Passport, Socket.io, Templating, etc...
